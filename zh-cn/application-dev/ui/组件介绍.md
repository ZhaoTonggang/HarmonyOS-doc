# 组件介绍<a name="ZH-CN_TOPIC_0000001063340551"></a>

-   [组件分类](#section154381954142018)

组件（Component）是构建页面的核心，每个组件通过对数据和方法的简单封装，实现独立的可视、可交互功能单元。组件之间相互独立，随取随用，也可以在需求相同的地方重复使用。
